Xbox Controller API v1.0 - Windows Package

This package contains the Xbox Controller API compiled for Windows.

Files included:
- xbox_controller_api.exe: Main executable
- SDL3.dll: SDL3 library for gamepad support
- libgcc_s_seh-1.dll: GCC runtime library
- libstdc++-6.dll: C++ standard library

Installation:
1. Extract all files to a folder
2. Run xbox_controller_api.exe
3. Connect your Xbox controller
4. The program will automatically detect and configure your controller

Configuration:
- A configuration file 'controller_config.txt' will be created automatically
- You can edit this file to customize button mappings and sensitivity

Controls:
- Left stick: Mouse movement
- Right stick: Scroll wheel
- A button: Left mouse click
- B button: Right mouse click
- X button: Play/Pause media
- Y button: Voice input (Win+H)
- Left Shoulder: Alt+Tab
- Right Shoulder: Sensitivity adjustment
- Start button: Exit program
- D-pad Up/Down: Adjust mouse sensitivity
- D-pad Left/Right: Adjust scroll sensitivity

Requirements:
- Windows 10 or later
- Xbox Controller (wired or wireless)
- Administrative privileges may be required for input simulation
